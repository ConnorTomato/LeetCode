#### [31. 下一个排列](https://leetcode-cn.com/problems/next-permutation/)

> - 执行用时：1 ms, 在所有 Java 提交中击败了98.25%的用户

```java
class Solution {
    public void nextPermutation(int[] nums) {
        int len = nums.length;
        for(int i = len - 1; i > 0; i--){
            if(nums[i] > nums[i - 1]){
                Arrays.sort(nums, i, len);
                for(int j = i; j < len; j++){
                    if(nums[j] > nums[i - 1]){
                        swap(nums, j, i - 1);
                        return;
                    }
                }
            }
        }
        Arrays.sort(nums);
    }
    private void swap(int[] nums, int i, int j){
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}
```

> - 字节变式：上一个排列
> - 我这个实现的比较复杂，其思路是，从头找到第一个比前面大的数，降序排列 （i + 1， len)

```java
    public static Integer[] helper(int[] nums){
        int len = nums.length;
        Integer[] arr = new Integer[len];
        for(int i = 0; i < len; i++){
            arr[i] = nums[i];
        }
        for(int i = 1; i < len; i++){
            if(arr[i] > arr[i - 1]){
                Arrays.sort(arr, i + 1, len,  Collections.reverseOrder());
                for(int j = i; j < len; j++){
                    if(arr[j] < arr[i]){
                        Integer temp = arr[j];
                        arr[j] = arr[i];
                        arr[i] = temp;
                        return arr;
                    }
                }
            }
        }
        return new Integer[]{-1};
    }
```
