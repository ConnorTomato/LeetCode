#### [231. 2 的幂](https://leetcode-cn.com/problems/power-of-two/)

> - 执行用时：1 ms, 在所有 Java 提交中击败了100.00%的用户

```java
class Solution {
    public boolean isPowerOfTwo(int n) {
        if(n <= 0){
            return false;
        }
        if(n == 1){
            return true;
        }
        return (n & 1) == 0 && isPowerOfTwo(n >>> 1);
    }
}

class Solution {
    public boolean isPowerOfTwo(int n) {
        if(n <= 0){
            return false;
        }
        if((n & n - 1) == 0){
            return true;
        }
        return false;
    }
}
```
