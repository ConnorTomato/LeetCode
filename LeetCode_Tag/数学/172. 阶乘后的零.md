#### [172. 阶乘后的零](https://leetcode-cn.com/problems/factorial-trailing-zeroes/)

``` java 
class Solution {
    public int trailingZeroes(int n) {
        int res = 0;
        for(int i = 2; i <= n; i++){
            int num = i;
            while(num != 0 && num % 5 == 0){
                res++;
                num /= 5;
            }
        }
        return res;
    }
}
```
