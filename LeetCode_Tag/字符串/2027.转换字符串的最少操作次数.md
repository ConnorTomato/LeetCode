####[2027. 转换字符串的最少操作次数](https://leetcode.cn/problems/minimum-moves-to-convert-string/)

``` java
class Solution {
    public int minimumMoves(String s) {
        int len = s.length();
        int res = 0;
        int index = 0;
        while (index < len) {
            if (s.charAt(index) == 'X') {
                res++;
                index += 3;
            } else {
                index++;
            }
        }
        return res;
    }
}
```
