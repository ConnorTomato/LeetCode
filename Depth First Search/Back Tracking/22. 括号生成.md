#### [22. 括号生成](https://leetcode-cn.com/problems/generate-parentheses/)

> - 执行用时：1 ms, 在所有 Java 提交中击败了83.94%的用户

```java
class Solution {
    public List<String> generateParenthesis(int n) {
        List<String> res = new ArrayList<>();
        StringBuilder path = new StringBuilder();
        backtracking(n, res, path, 0, 0);
        return res;
    }
    private void backtracking(int n, List<String> res, StringBuilder path, int leftBracket, int rightBracket){
        if(leftBracket == n && rightBracket == n){
            res.add(path.toString());
            return;
        }
        if(leftBracket < n){
            path.append('(');
            backtracking(n, res, path, leftBracket + 1, rightBracket);
            path.deleteCharAt(path.length() - 1);
        }
        if(leftBracket > rightBracket){
            path.append(')');
            backtracking(n, res, path, leftBracket, rightBracket + 1);
            path.deleteCharAt(path.length() - 1);
        }
    }
}
```
