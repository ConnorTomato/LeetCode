#### [695. 岛屿的最大面积](https://leetcode-cn.com/problems/max-area-of-island/)

> - 执行用时：3 ms, 在所有 Java 提交中击败了52.10%的用户

```java
class Solution {
    public int maxAreaOfIsland(int[][] grid) {
        int m = grid.length;
        int n = grid[0].length;
        int res = 0;
        boolean[][] visited = new boolean[m][n];
        for(int i = 0; i < m; i++){
            for(int j = 0; j < n; j++){
                if(!visited[i][j] && grid[i][j] == 1){
                    res = Math.max(res, dfs(grid, m, n, visited, i, j));
                }
            }
        }
        return res;
    }
    private int dfs(int[][] grid, int m, int n, boolean[][] visited, int i, int j){
        if(i < 0 || i >= m || j < 0 || j >= n || visited[i][j] || grid[i][j] != 1){
            return 0;
        }
        visited[i][j] = true;
        int sum = dfs(grid, m, n, visited, i - 1, j)
                + dfs(grid, m, n, visited, i + 1, j)
                + dfs(grid, m, n, visited, i, j - 1)
                + dfs(grid, m, n, visited, i, j + 1);
        return sum + 1;
    }
}
```
