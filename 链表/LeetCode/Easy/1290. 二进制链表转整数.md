#### [1290. 二进制链表转整数](https://leetcode-cn.com/problems/convert-binary-number-in-a-linked-list-to-integer/)

> - 执行用时：0 ms, 在所有 Java 提交中击败了100.00%的用户

```java
class Solution {
    public int getDecimalValue(ListNode head) {
        int sum = 0;
        while(head != null){
            sum <<= 1;
            sum += head.val;
            head = head.next;
        }
        return sum;
    }
}
```
