

#### [143. 重排链表](https://leetcode-cn.com/problems/reorder-list/)

> - 寻找链表中点(快慢指针) + 链表逆序（反转链表） + 合并链表
> - 执行用时：2 ms, 在所有 Java 提交中击败了58.27%的用户

```java
class Solution {
    public void reorderList(ListNode head) {
        if(head.next == null){
            return;
        }
        // 寻找链表中点,链表 5，slow 最终应该在 2；
        ListNode fast = head, slow = head;
        while(fast.next != null && fast.next.next != null){
            fast = fast.next.next;
            slow = slow.next;
        }
        // 反转后半段链表
        ListNode reverseNode = slow.next;
        slow.next = null;
        reverseNode = reverse(reverseNode);

        // 合并链表，插入前半段间隙
        ListNode cur = head;
        while(cur != null && reverseNode != null){
            ListNode curNextNode = cur.next;
            ListNode movedNode = reverseNode;
            reverseNode = reverseNode.next;
            movedNode.next = curNextNode;
            cur.next = movedNode;
            cur = curNextNode;
        }
    }
    private ListNode reverse(ListNode head){
        ListNode pre = null, cur = head;
        while(cur != null){
            ListNode next = cur.next;
            cur.next = pre;
            pre = cur;
            cur = next;
        }
        return pre;
    }
}
```
