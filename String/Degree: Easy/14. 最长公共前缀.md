#### [14. 最长公共前缀](https://leetcode-cn.com/problems/longest-common-prefix/)

> - 执行用时：0 ms, 在所有 Java 提交中击败了100.00%的用户

```java
class Solution {
    public String longestCommonPrefix(String[] strs) {
        int len = strs.length;
        if(len == 0){
            return "";
        }
        String res = strs[0];
        for(String str : strs){
            while(!str.startsWith(res)){
                if(res.length() == 0){
                    return "";
                }
                //公共前缀不匹配就让它变短！
                res = res.substring(0, res.length() - 1);
            }
        }
        return res;
    }
}
```
